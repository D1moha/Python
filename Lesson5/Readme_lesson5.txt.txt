Задание:
Используя sqlite3, напишите код, который создает таблицу базы данных для датасета с CityId городов различных стран мира, 
приведенного в файле https://worldweather.wmo.int/en/json/full_city_list.txt. 
Предусмотреть функцию, позволяющую вносить изменения в базу данных. 
Продемонстрировать её работу.

Для запуска программы нужно запустить на выполнение файл Lesson5.py
Программа состоит из функций:
1. Функция create_new_table() Создаёт таблицу в базе данных 
2. Функция full_city_list() Добавляет данные из файла в базу данных
3. Функция insert_city_list() Добавляет новые данные в базу
4. Функция def update_city_list() Обновляет данные в базе по указанному коду
5. Функция delete_city_list() Удаляет данные в базе по указанному коду
При запуске файла программа выполнит следующее:
1. Подключится к базе sitylist.db и создаст в ней новую таблицу NewTable (если такой ранее не было)
2. Перенесёт данные из файла в базу (если данные до этого не были перенесены)
3. Предложит пользователю добавить новые данные в базу. При вводе "Да", программа попросит пользователя ввести Страну, Город и Код города, после чего добавит новую запись в таблицу с этими данными. При вводе "Нет", программа перейдёт к следующему шагу.
4. Предложит пользователю обновить данные в базе. При вводе "Да", программа попросит пользователя ввести Код города, выполнится проверка на наличие в базе города с таким кодом, если город есть, тогда программа выведет Город и Страну, и попросит пользователя подтвердить обновление. После подтверждения от пользователя, программа предложит ввести новую страну и город. После чего обновит данные в базе и выведет об этом уведомление. Если пользователь откажется от обновлений, либо города с таким кодом не будет, либо пользователь не подтвердит обновление конкретного города, тогда программа перейдёт к следующему шагу.
5. Предложит пользователю удалить данные из базы. При вводе "Да", программа попросит пользователя ввести Код города, выполнится проверка на наличие в базе города с таким кодом, если город есть, тогда программа выведет Город и попросит пользователя подтвердить удаление. После подтверждения от пользователя, программа удалит данные из базы и выведет об этом уведомление. Если пользователь откажется от удаления, либо города с таким кодом не будет, либо пользователь не подтвердит удаление конкретного города, тогда программа закончит свою работу.
